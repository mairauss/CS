# CS
Smart neighborhoods

## How to run:
1) Install needed dependencies: 
```zsh
pip install -r requirements.txt
```
2) After run the main.py:
```zsh
python src/main.py
```
or
```zsh
python3 src/main.py
```
## How to run using Docker:
Use the following command:

```zsh
docker-compose -f stack.yml up
```
